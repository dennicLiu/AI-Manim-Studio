## 🎬 产品简介

AI Manim Studio 是一款专业的数学教学动画视频生成工具。只需输入主题，AI 会自动与您对话确认视频方向，然后生成精美的 Manim 动画脚本并渲染成视频。让复杂的数学概念通过生动有趣的动画变得简单易懂。

---

## ✨ 核心功能

### 1. 智能对话式视频策划
- **多轮对话确认**：AI 通过 2-3 轮对话了解您的需求
- **自动提问**：AI 会询问目标受众、内容重点、风格偏好等
- **个性化定制**：根据对话生成符合您需求的视频脚本

### 2. 一键生成动画脚本
- **自动生成**：AI 根据对话内容自动生成完整的 Manim 动画脚本
- **代码优化**：自动修复常见错误，确保脚本可正常运行
- **专业代码**：生成的代码符合 Manim 最佳实践

### 3. 多种渲染选项
- **质量选择**：
  - 低质量（480p）- 快速预览
  - 中等质量（720p）- 平衡效果
  - 高质量（1080p）- 专业制作
- **可选渲染**：可选择仅生成脚本，或自动渲染成视频

### 4. 支持的教学主题
- ✅ 几何图形（圆、三角形、正方形等）
- ✅ 函数图像（二次函数、三角函数等）
- ✅ 物理现象（自由落体、运动等）
- ✅ 数学公式推导
- ✅ 任何数学概念的可视化

### 5. 友好的用户界面
- **现代化设计**：类似微信聊天的对话界面
- **步骤清晰**：三个步骤轻松完成视频制作
  - 步骤1：主题确认（多轮对话）
  - 步骤2：脚本生成与渲染
  - 步骤3：预览与下载
- **实时反馈**：清晰的状态提示和进度显示

### 典型应用场景

#### 1. 圆的面积教学视频
- **效果**：从生活实例（披萨、盘子）引入圆的概念
- **动画**：圆形分解、公式推导、实例计算
- **时长**：2-3 分钟
- **适用**：小学数学教学

- **效果**：函数图像动态绘制，展示开口方向、顶点、对称轴
- **动画**：坐标轴建立、点动画、曲线绘制
- **适用**：初中数学教学

- **效果**：从直角三角形到单位圆，直观展示 sin、cos、tan
- **动画**：图形变换、角度变化、数值对应
- **适用**：高中数学教学

#### 4. 自由落体运动
- **效果**：物体下落过程可视化，展示速度、加速度变化
- **动画**：物体运动轨迹、速度矢量、公式推导
- **适用**：物理教学
